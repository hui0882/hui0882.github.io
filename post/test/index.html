文章结构

  1. Front Matter (头部信息)

    • title: 文章标题。
    • date: 文章的发布时间。格式为 YYYY-MM-DDTHH:MM:SS+时区偏移量。例如，2025-03-24T13:21:00+08:00。
    • draft: 是否为草稿。设置为 true 则该文章不会被生成到最终网站中；设置为 false 则会被发布。
    • tags: 标签列表，帮助分类和搜索文章。
    • categories: 分类列表，类似于标签，但通常用于更广泛的分类。
    • author: 文章作者的名字。
    • description: 文章的简短描述，通常用于 SEO 和社交分享。
    • cover: 封面图片的路径。此字段用于指定文章封面图片的位置。

  2. 段落
    正文内容直接写成段落即可。每一段之间空一行分隔。

  3. 列表

    • 无序列表：使用 - 或 *。 
      - 列表项 1
- 列表项 2
    • 有序列表：使用数字后跟点号。 
      1. 第一项
2. 第二项

  4. 图片
    插入图片时，使用 Markdown 的图片语法 ![替代文字](图片路径)。

  5. 链接
    插入链接时，使用 [链接文字](URL)。
    例如：访问 Hugo 官网。

  6. 加粗和斜体

    • 加粗：**加粗文字** 或 __加粗文字__。
    • 斜体：*斜体文字* 或 _斜体文字_。

总结

通过本文，我们学习了如何在 Hugo 中编写一篇带封面和插图的文章。以下是关键点总结：

  • Front Matter (头部信息)：包含文章的基本信息，如标题、发布时间、标签、类别等。
  • 封面图片：通过 cover 元数据字段指定封面图片路径，或者手动插入图片。
  • 正文图片：将图片放在 static/images 文件夹中，并使用 Markdown 语法插入。
  • 格式化内容：使用 Markdown 提供的语法格式化文本，如标题、列表、加粗等。
